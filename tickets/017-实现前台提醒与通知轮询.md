# 实现前台提醒与通知轮询

## User Story

作为用户，我希望应用在打开时轮询提醒和通知，以便无需手动刷新即可看到更新。

## Acceptance Criteria

### Scenario 1: 应用打开时轮询提醒

Given 应用处于前台
When 应用打开
Then 轮询到期提醒

### Scenario 2: 应用恢复时轮询通知

Given 应用曾被置于后台
When 应用返回前台
Then 轮询新通知

### Scenario 3: 可见性变化时轮询

Given 页面可见
When 可见性变为可见
Then 触发轮询
