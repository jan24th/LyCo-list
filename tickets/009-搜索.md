# 搜索

## User Story

As a 用户，I want 按标题和备注搜索任务，So that 我可以不浏览列表就快速找到特定项目。

## Acceptance Criteria

### Scenario 1: 按标题搜索

Given 数据库中存在任务
When 用户输入匹配任务标题的关键词
Then 搜索结果返回匹配的任务

### Scenario 2: 按备注搜索

Given 任务包含备注
When 用户输入出现在任务备注中的关键词
Then 搜索结果返回匹配的任务

### Scenario 3: 空搜索结果

Given 没有任务匹配搜索关键词
When 用户提交搜索
Then 显示空状态提示没有结果

### Scenario 4: 清空搜索

Given 用户已输入搜索词
When 他们清空搜索输入
Then 恢复之前的列表视图

### Scenario 5: 搜索性能

Given 存在数千条任务
When 用户输入搜索查询
Then 由于使用 IndexedDB 索引，结果在合理时间内返回
