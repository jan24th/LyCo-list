# 智能列表与搜索

## User Story

As a 用户，I want 自动聚合的任务视图和搜索能力，So that 我可以快速查看今天、计划、已标记等任务并搜索内容。

## Acceptance Criteria

### Scenario 1: 显示今天列表

Given 存在今天到期的未完成任务
When 用户打开"今天"智能列表
Then 显示所有今天到期的任务

### Scenario 2: 显示计划列表

Given 存在有截止日期的未完成任务
When 用户打开"计划"智能列表
Then 显示所有已安排的任务，按日期排序

### Scenario 3: 显示已标记列表

Given 存在已标记任务
When 用户打开"已标记"智能列表
Then 显示所有已标记任务

### Scenario 4: 显示已完成列表

Given 存在已完成任务
When 用户打开"已完成"智能列表
Then 显示所有已完成任务

### Scenario 5: 搜索任务

Given 用户输入关键词
When 调用搜索接口
Then 结果列表显示匹配的任务

### Scenario 6: TDD 开发与 100% 覆盖率

Given 项目要求 TDD
When 为智能列表和搜索组件编写测试
Then 测试先失败，实现后通过，覆盖率达到 100%
