# 截止日期与重复提醒

## User Story

As a 用户，I want 为任务设置截止日期和重复提醒，So that 我可以跟踪截止期限和重复习惯。

## Acceptance Criteria

### Scenario 1: 设置截止日期和时间

Given 用户编辑任务
When 选择 dueDate 和 dueTime
Then 任务保存时以 UTC 存储并在前端以本地时区显示

### Scenario 2: 设置重复提醒

Given 用户创建提醒
When 选择 daily/weekly/biweekly/monthly/yearly/weekdays 频率
Then 后端计算并保存 nextTriggerAt

### Scenario 3: 重复任务完成后推进

Given 任务有重复提醒
When 用户标记任务完成
Then 任务保持活跃，dueDate 和提醒推进到下一周期

### Scenario 4: 一次性提醒完成后禁用

Given 任务有一次性提醒
When 提醒触发或任务完成
Then 提醒被禁用

### Scenario 5: 在智能列表中显示到期任务

Given 任务有 dueDate
When 用户查看"今天"或"计划"列表
Then 任务按本地时区正确出现在对应列表

### Scenario 6: TDD 开发与 100% 覆盖率

Given 项目要求 TDD
When 为提醒计算和调度逻辑编写测试
Then 测试先失败，实现后通过，覆盖率达到 100%
