# 实现通知查询、标记已读和 TTL

## User Story

作为用户，我希望查询通知、标记已读并让通知过期，以便收件箱保持整洁。

## Acceptance Criteria

### Scenario 1: 查询通知

Given 已认证用户拥有通知
When 用户查询通知
Then 返回分页结果

### Scenario 2: 标记通知为已读

Given 一个未读通知
When 用户将其标记为已读
Then 通知的已读状态被更新

### Scenario 3: TTL 过期

Given 通知的 TTL 已过期
When 清理运行
Then 通知被删除
