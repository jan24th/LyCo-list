# 前端脚手架 React + Vite + TanStack

## User Story

As a 前端开发者，I want 一个现代 React SPA 基础，So that 可以在此之上构建 PWA 待办应用界面。

## Acceptance Criteria

### Scenario 1: 初始化 Vite 项目

Given `apps/web` 目录已存在
When 使用 React + TypeScript 模板初始化 Vite
Then 运行 `bun dev` 后开发服务器可访问

### Scenario 2: 配置 Tailwind CSS

Given Vite 项目已创建
When 安装并配置 Tailwind CSS
Then 组件中可以使用 Tailwind 工具类

### Scenario 3: 配置 TanStack Router

Given 前端项目已搭建
When 安装并配置 TanStack Router
Then 应用支持客户端路由导航

### Scenario 4: 配置 TanStack Query

Given 前端项目已搭建
When 安装并配置 TanStack Query
Then 组件可以通过 hooks 调用 API 并缓存数据

### Scenario 5: 配置 TanStack Store

Given 前端项目已搭建
When 安装并配置 TanStack Store
Then 可以在组件间共享临时 UI 状态

### Scenario 6: 配置 TanStack Form

Given 前端项目已搭建
When 安装 TanStack Form
Then 可以创建带校验和状态管理的表单

### Scenario 7: 配置测试环境

Given 项目要求 TDD
When 在 `apps/web` 中配置 Vitest、Testing Library 和 MSW
Then 可以运行组件和逻辑测试
